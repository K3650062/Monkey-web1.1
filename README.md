# Jungles & Monkeys

A modern, immersive website about jungles and monkey species.

## Features
- **Interactive Map**: Explore monkey habitats globally.
- **Species Explorer**: Search and filter primate species.
- **Gamified Learning**: Play "Match the Monkey" to learn.
- **Contribution**: Submit your own sightings.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS (v4)
- Framer Motion
- Leaflet (React Leaflet)
- TypeScript

## Getting Started

1. **Install Dependencies**:
   ```bash
   npm install
   ```

2. **Run Development Server**:
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) to view it in the browser.

3. **Build for Production**:
   ```bash
   npm run build
   npm start
   ```

## Project Structure
- `src/app`: Next.js App Router pages.
- `src/components`: Reusable UI components.
- `src/features`: Feature-specific components (Map, Species, Games).
- `src/lib`: Data and utilities.

## Deployment
Deploy easily on Vercel:
1. Push to GitHub.
2. Import project in Vercel.
3. Deploy!
